# Molin Dashboard And Docs Map

This file gives a high-level picture of the Molin product surface without implementation details.

## Dashboard Areas

The Molin dashboard generally revolves around these areas:

- inbox and conversations
- AI setup and chatbot customization
- knowledge sources such as products, documents, and FAQs
- analytics and reporting
- integrations
- settings, billing, plans, and usage

## Common Chatbot Areas

When users ask where to manage the chatbot, the common buckets are:

- personality or instructions
- design and appearance
- publishing or embed setup
- actions or automations

## Knowledge / Training Areas

When users ask how the AI learns, the common buckets are:

- products
- documents
- FAQs
- web pages or uploaded content

## Settings Areas

Common settings categories include:

- profile
- chatbots
- members or team management
- billing
- plans
- usage

## Docs Structure

The docs are organized around a few main themes:

- getting started
- platform integrations
- feeds and product sources
- API reference
- tracking
- legal and policy pages

## Docs Guidance

- If the question is “where do I learn about setup?”, point to getting started or platform docs.
- If the question is about integrations, point to the relevant platform or feed area.
- If the question is technical, point to the API reference.
- If the question is about compliance or policies, point to the legal docs.
